# risk_monte_carlo
The objective was to analyze over six decades of historical stock market data to understand a stock's performance, quantify its risk characteristics, simulate future price behavior using stochastic models, and validate simulation results against real market data.

The two files - brownianmotion.ipynb and montecarlo-experiment.ipynb are tutorial files where I was working through classic examples of how to use these simulations.

# The main project-

Contained in quant_analysis_simulation.ipynb:

Developed a quantitative risk analysis framework for historical stock data (1962–present) using Python. Implemented return and volatility analysis, Sharpe/Sortino ratios, Value at Risk (VaR), Conditional VaR (CVaR), technical indicators, and Monte Carlo (10,000-path Geometric Brownian Motion) simulations. Validated probabilistic forecasts against out-of-sample market data using MAE, RMSE, MAPE, and confidence interval coverage.

Dataset used: https://www.kaggle.com/datasets/jakewright/9000-tickers-of-stock-market-data-full-history/data
